## 1. 추진 목적

### 1.1 사업 제안 배경

교육 콘텐츠 신규 기획 시, 기획자는 주제 선정 단계에서 다음과 같은 **구조적 문제**를 반복적으로 경험합니다.

```
┌──────────────────────────────────────────────────────────────────────────┐
│                     📍 기획자가 겪는 4대 Pain Point                        │
├──────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│     ❶ 트렌드 판단 불확실            ❷ 정량적 근거 부족                        │
│     ┌───────────────┐              ┌───────────────┐                     │
│     │      🎯       │              │      ❓       │                    │
│     │   감각 의존    │              │  "왜 이 주제  │                       │
│     │ 커뮤니티 반응  │              │   인가요?"    │                       │
│     └───────────────┘              └───────────────┘                     │
│                                                                          │
│     ❸ 우선순위 합의 지연            ❹ 기획 리스크                            │
│     ┌───────────────┐              ┌───────────────┐                     │
│     │    🗣️ 🗣️     │              │      📉       │                    │
│     │   A vs B      │              │   반짝 유행   │                      │
│     │   의견 대립    │              │   장기 성과 ↓ │                      │
│     └───────────────┘              └───────────────┘                     │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘
```

이러한 문제는 기획자 개인의 역량 문제가 아닌, **의사결정을 지원하는 구조화된 도구의 부재**에서 비롯된 문제입니다.

## 2. 문제 적정성

### 2.1 현행 방식의 한계

| 항목 | 현행 방식 | 한계점 |
|:---|:---|:---|
| **트렌드 판단** | Google 개별 검색 | 주관적, 일관성 부족 |
| **비교 기준** | 스크린샷 수집 | 정량 비교 불가 |
| **회의 설득** | 경험 기반 발언 | 합의 지연, 반복 논의 |
| **리스크 관리** | 사후 확인 | 반짝 유행 선별 불가 |

**실제 시나리오 예시:**
> "ChatGPT 강의" vs "AI 활용" vs "생성형 AI" 중 어떤 키워드로 기획할지 판단할 때,
> 각 키워드의 성장률, 안정성, 학습 수요 전환 신호를 비교할 객관적 기준이 부재합니다.

### 2.2 문제 적정성 판단

교육 콘텐츠는 단순 정보가 아닌 **하나의 상품**입니다.
주제 선정 단계의 판단 오류는 **성과·브랜드·신뢰도**에 직접적인 영향을 미칩니다.

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│    📌 주제 선정 = 상품 기획의 시작점                           │
│                                                             │
│    잘못된 주제 선정 → 리소스 낭비 → 성과 미달 → 신뢰도 하락        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 3. 추진 내용 및 솔루션

### 3.1 EduTrend Finder 개요

EduTrend Finder는 **Web 검색(정보 탐색 수요)**과 **YouTube 검색(학습 실행 수요)**를 교차 분석하여
교육 콘텐츠로의 전환 가능성 신호를 포착하는 **기획 의사결정 보조 도구**입니다.

```
┌──────────────────────────────────────────────────────────────────────────┐
│                      EduTrend Finder 분석 흐름                            │
├──────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│                        ┌────────────────┐                                │
│                        │  키워드 입력    │                                 │
│                        └───────┬────────┘                                │
│                                ↓                                         │
│              ┌─────────────────────────────────────┐                     │
│              │        데이터 수집 (병렬 처리)        │                      │
│              │   ┌──────────┐     ┌──────────┐    │                      │
│              │   │ Web 검색  │     │YouTube검색│    │                     │
│              │   │(탐색 수요)│     │(학습 수요)│    │                      │
│              │   └─────┬────┘     └─────┬────┘    │                      │
│              └─────────┼────────────────┼─────────┘                      │
│                        ↓                ↓                                │
│              ┌─────────────────────────────────────┐                     │
│              │           교차 분석 엔진             │                      │
│              │  • 성장률 계산 (초기 vs 최근 30%)   │                        │
│              │  • 변동성 분석 (변동계수 CV)        │                        │
│              │  • 상관계수 산출 (Web-YouTube)      │                       │
│              └──────────────┬──────────────────────┘                     │
│                             ↓                                            │
│              ┌─────────────────────────────────────┐                     │
│              │           진단 엔진                  │                     │
│              │  • 시장 단계 판정 (5단계)           │                       │
│              │  • 트렌드 성격 분류 (4유형)         │                        │
│              │  • 교차 신호 해석 (6패턴)           │                        │
│              └──────────────┬──────────────────────┘                     │
│                             ↓                                            │
│              ┌─────────────────────────────────────┐                     │
│              │           기획 인사이트              │                      │
│              │   🚀 신규 기획  |  🔄 리뉴얼  |  ⛔ 보류  │                 │
│              └─────────────────────────────────────┘                     │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘
```

### 3.2 핵심 분석 지표

| 지표 | 산출 방식 | 의미 |
|:---|:---|:---|
| **성장률** | (최근 30% 평균 - 초기 30% 평균) / 초기 30% 평균 × 100 | 트렌드 방향성 |
| **변동계수(CV)** | 표준편차 / 평균 | 안정성 (낮을수록 안정) |
| **상관계수** | Web-YouTube Pearson 상관 | 학습 수요 전환 신호 |

### 3.3 진단 로직: 시장 단계 판정

| 단계 | 조건 | 기획 시사점 |
|:---|:---|:---|
| 🌱 **도입기** | 성장률 > 50%, 최근 관심도 < 30 | 선점 기회, 리스크 존재 |
| 📈 **성장기** | 성장률 > 20%, 최근 관심도 ≥ 30 | **적극 진입 권장** |
| 🏔️ **성숙기** | \|성장률\| < 15%, 최근 관심도 ≥ 50 | 차별화 전략 필요 |
| 📉 **쇠퇴기** | 성장률 < -10% | 철수 또는 리뉴얼 검토 |
| 🔄 **전환기** | 기타 | 추가 모니터링 필요 |

### 3.4 진단 로직: Web-YouTube 교차 신호

| 패턴 | 해석 | 기획 액션 |
|:---|:---|:---|
| Web↑ YouTube↑ | 강한 학습 수요 | **🚀 교육 상품 적기** |
| Web→ YouTube↑ | 학습 채널 이동 | **🎬 영상 콘텐츠 우선** |
| Web↑ YouTube→ | 정보 탐색 단계 | 📊 시장 모니터링 지속 |
| Web↓ YouTube↓ | 관심 하락 | ⛔ 진입 재검토 |

## 4. 기대 효과

### 4.1 Before → After 비교

| 항목 | Before | After |
|:---|:---|:---|
| **트렌드 조사** | 개별 Google 검색 + 스크린샷 | 30개 키워드 자동 분석 + 성장률 자동 계산 |
| **비교 기준** | "더 뜨는 것 같다" | 성장률 45% vs 12%, 상관계수 0.7 vs 0.3 |
| **회의 준비** | 약 3시간 리서치 | 약 30분 내 리포트 생성 |
| **의사결정** | 의견 대립 → 반복 회의 | 데이터 기반 1회 합의 |
| **리스크 관리** | 사후 확인 | 사전 변동성 체크 (CV 지표) |

### 4.2 핵심 성과지표(KPI)

| 구분 | 지표 | 측정 방법 |
|:---|:---|:---|
| **기획 효율** | 후보 키워드 리서치 소요 시간 | 목표: 3시간 → 30분 |
| **의사결정** | 주제 확정까지 회의 횟수 | 목표: 3회 → 1회 |
| **품질** | 데이터 기반 선정 주제 성과 | 감각 기반 대비 비교 분석 |

## 8. 결론

EduTrend Finder는 교육 콘텐츠 기획자의 주제 선정 판단을
**기획자 개인의 감각**에서 **조직의 데이터 자산**으로 확장하는 도구입니다.

본 제안은 에이블런의 핵심 업무인 **'교육 상품 시장성 분석 및 검증'**에
**데이터 기반 의사결정 체계**를 도입하는 방안입니다.

EduTrend Finder는 기획자의 감각을 대체하는 것이 아니라,
**감각을 데이터로 검증하고 설명할 수 있게 돕는 도구**로서 가치를 가집니다.

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   "왜 이 주제를 선택했는가?"                                   │
│                                                             │
│   → "성장률 45%, Web-YouTube 동반 상승, 상관계수 0.72"          │
│                                                             │
│   감각에서 데이터로, 개인에서 조직으로.                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
