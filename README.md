# EduTrend Finder

### 데이터 기반 교육 트렌드 분석 및 기획 의사결정 보조 도구

> **EduTrend Finder**는 Web 검색과 YouTube 검색 데이터를 교차 분석해,
> 교육 콘텐츠·사업 기획자가 '감'이 아닌 '설명 가능한 데이터'를 기반으로
> **교육 상품 론칭 우선순위를 결정할 수 있게 돕는 의사결정 보조 도구**입니다.

## 문제 인식 (Problem Statement)

교육 시장에서 "요즘 뜨는 교육 주제"를 판단하고 실제 상품으로 론칭할 때, 기획자의 감각이나 커뮤니티 분위기에 의존하는 경우가 많습니다.

**이로 인해 발생하는 문제:**

| 문제 | 결과 |
|------|------|
| 단기 이슈 vs 지속 트렌드 구분 실패 | 반짝 유행에 리소스 낭비 |
| "왜 이 주제인가?"에 대한 근거 부재 | 기획 회의에서 설득력 부족 |
| 후보 간 우선순위 판단 기준 부재 | 의사결정 지연, 시장 진입 타이밍 놓침 |

**EduTrend Finder는 이 문제를 해결합니다:**
- 검색 데이터 기반의 **정량적 판단 체계** 제공
- Web-YouTube 교차 분석으로 **학습 수요 전환 신호** 포착
- 자동화된 진단 로직으로 **즉시 활용 가능한 인사이트** 도출

## 핵심 기능

### 1. 트렌드 대시보드
30개 교육 키워드의 실시간 트렌드를 한눈에 파악

- 성장률 기반 자동 정렬
- 시장 단계 라벨링 (도입기/성장기/성숙기/쇠퇴기)
- 클릭 시 상세 분석 페이지 이동

### 2. 키워드 비교 분석
최대 5개 키워드를 동일 차트에서 직접 비교

- Web/YouTube 데이터 분리 또는 통합 보기
- 이동 평균(7일) 적용으로 노이즈 제거
- Min-Max 정규화 옵션으로 공정한 추세 비교

### 3. 인사이트 리포트
기획 회의에 바로 활용할 수 있는 종합 분석 리포트

- 우선순위 키워드 자동 추천
- Web-YouTube 상관관계 분석
- 리스크 요인 및 권장 액션 제시

---

## 분석 파이프라인

```
[1] 키워드 풀 정의
     └─→ 교육 도메인 키워드 30개 사전 정의

[2] 데이터 수집 (병렬 처리)
     ├─→ Web 검색 트렌드 수집
     └─→ YouTube 검색 트렌드 수집

[3] 전처리 및 보정
     ├─→ 7일 이동 평균 적용 (노이즈 제거)
     └─→ Min-Max 정규화 (키워드 간 공정 비교)

[4] 지표 계산
     ├─→ 성장률 (초기 30% vs 최근 30% 평균 비교)
     ├─→ 변동성 (변동계수 CV)
     └─→ Web-YouTube 상관계수 (Pearson)

[5] 진단 로직 적용
     ├─→ 시장 단계 판정 (도입기/성장기/성숙기/쇠퇴기)
     ├─→ 트렌드 분류 (지속 성장 vs 일시적 급등)
     └─→ 교차 신호 해석 (Web-YouTube 패턴 매칭)

[6] 전략 인사이트 도출
     ├─→ 우선순위 키워드 추천
     ├─→ 기획 액션 제안
     └─→ 리스크 요인 명시
```

## 진단 로직 상세

### 시장 단계 판정

| 단계 | 조건 | 의미 |
|------|------|------|
| 도입기 | 성장률 > 50%, 최근 관심도 < 30 | 초기 시장, 선점 기회 |
| 성장기 | 성장률 > 20%, 최근 관심도 >= 30 | 수요 확대 중, 적극 진입 |
| 성숙기 | \|성장률\| < 15%, 최근 관심도 >= 50 | 안정적 수요, 차별화 필요 |
| 쇠퇴기 | 성장률 < -10% | 수요 감소, 철수 검토 |

### 트렌드 성격 분류

| 분류 | 조건 | 기획 액션 |
|------|------|-----------|
| 지속 성장 | 기울기 > 0.3, 변동계수 < 0.4 | 신규 기획 적극 검토 |
| 일시적 급등 | 변동계수 > 0.5, 최대/평균 > 2.0 | 단기 테스트 후 결정 |
| 안정적 유지 | \|기울기\| < 0.1, 변동계수 < 0.3 | 기존 콘텐츠 리뉴얼 |
| 급등 후 하락 | 최근 피크 + 음의 기울기 | 진입 보류 권장 |

### Web-YouTube 교차 신호

| 패턴 | 해석 | 시사점 |
|------|------|--------|
| Web↑ YouTube↑ | 강한 학습 수요 | 교육 상품 적기 |
| Web→ YouTube↑ | 학습 채널 이동 | 영상 콘텐츠 우선 |
| Web↑ YouTube→ | 정보 탐색 단계 | 시장 모니터링 지속 |
| Web↓ YouTube↓ | 관심 하락 | 진입 재검토 |

## 활용 시나리오

| 시나리오 | 활용 방법 |
|----------|-----------|
| **신규 강의 기획** | 성장기 + 지속 성장 키워드 중 Web-YouTube 상관 높은 주제 선정 |
| **기존 강의 리뉴얼** | 성숙기 키워드 중 최근 관심도 상승 조짐 있는 주제 우선 |
| **마케팅 메시지** | YouTube 성장률 > Web 성장률인 키워드는 "영상으로 배우세요" 강조 |
| **리소스 배분** | 쇠퇴기 키워드는 유지만, 성장기 키워드에 신규 리소스 집중 |

## 한계 및 향후 계획

### 현재 한계
- Google Trends는 **상대적 검색 관심도(0-100)**를 제공하며, 절대 검색량이 아님
- API Rate Limit으로 인해 실시간 업데이트에 제약
- 키워드 풀이 사전 정의된 30개로 고정

### 향후 로드맵

| Phase | 기능 | 목표 |
|-------|------|------|
| **Phase 2** | 키워드 자동 추천 | 연관 키워드 자동 발굴 |
| **Phase 2** | 리포트 PDF 다운로드 | 회의 자료 즉시 활용 |
| **Phase 3** | 경쟁사 콘텐츠 분석 | 시장 포지셔닝 인사이트 |
| **Phase 3** | 알림 기능 | 특정 키워드 급등 시 알림 |
